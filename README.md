# homebrew-openloco

Personal Homebrew tap for OpenLoco formula.

## Installation

```bash
brew tap jpdck/openloco
brew install openloco
```

## Usage

Run from the command line:

```bash
openloco
```

Or add to your Applications folder:

```bash
ln -s $(brew --prefix openloco)/OpenLoco.app /Applications/OpenLoco.app
```

## Notes

This works on my machine (mbp m4 running tahoe). If you find any issues, please open an issue in this repository so we can address them.
